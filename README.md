#  Real-Time Weather Application

A sleek, responsive weather application that provides real-time meteorological data for cities worldwide. Built to demonstrate proficiency in API integration, asynchronous JavaScript, and modern UI design.

** [Live Demo](https://weather-iota-indol-30.vercel.app/)**

## Features

* **Real-Time Data:** Fetches current temperature, humidity, and wind speed using the OpenWeatherMap API.
* **Dynamic Weather Icons:** Visuals update automatically based on weather conditions (Clear, Clouds, Rain, etc.).
* **Location Search:** Search for any city globally to get instant weather updates.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
* **Interactive UI:** Clean and minimalist interface with smooth transitions.

## Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **API:** OpenWeatherMap API
* **Deployment:** Vercel
